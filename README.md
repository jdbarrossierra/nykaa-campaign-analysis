# Proyecto 8: Análisis de Campañas Digitales Nykaa

Dashboard interactivo en Excel + Power Query usando dataset real de Kaggle (expandido a ~111k filas tras normalización de canales múltiples).

## Preguntas clave respondidas
1. Tipo de campaña con mayor ROI → Social Media (promedio 2.74)  
2. Canal con mejor tasa de conversión → Instagram (~1045 conversiones promedio)  
3. Costo de adquisición más bajo → Email (~367) | Más alto → YouTube (~385)  
4. Período de mayores ingresos → Noviembre-Diciembre 2024 (temporada navideña)  
5. Segmento más rentable → Premium Shoppers (líder en ingresos y conversiones)  
6. Relación Engagement Score vs Conversiones → Correlación positiva moderada (R² ≈ 0.37)
